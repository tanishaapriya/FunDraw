# 🎨 FUNRAW – Air Drawing using Hand Gestures

A real-time computer vision project that allows users to draw in the air using hand gestures. Built using OpenCV and MediaPipe.

---

## 🚀 Features

* ✋ Hand gesture recognition
* 🎨 Air drawing using index finger
* 🌈 Multi-color palette selection
* 💾 Save drawings using gestures
* 🧽 Erase canvas using gestures
* ✋ Stop drawing mode
* 👤 Face detection activation

---

## 🧠 Tech Stack

* Python
* OpenCV
* MediaPipe
* NumPy

---

## 🎮 Gesture Controls

| Gesture       | Action |
| ------------- | ------ |
| ☝️ 1 Finger   | Draw   |
| ✌️ 2 Fingers  | Pause  |
| 🤟 3 Fingers  | Erase  |


---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
